1 Introduction

Until now, we have learnt general concepts about geospatial visualization in independent lessons. The modular approach of {ggplot2} allows to successively add all of them in different layers.

However, these enriched thematic maps also require to contain text and labels referring to specific places or regions, and important map elements like scale bars and a north arrow, as will be illustrated in this part.

In this lesson, we will learn to use functions from the {ggspatial} and {ggsflabel} packages to add all of these additional elements to our maps!

2 Learning objectives

  1. Access hospital data coordinates using the {afrihealthsites} package.

  2. Add text and labels to ggplot maps using the {ggsflabel} package.

  3. Add arrow and scale annotations in ggplot maps using the {ggspatial} package.

â€¢ We can use {ggrepel} arguments to improve its output:

â€¢ The size (argument size);

â€¢ The type of font (fontface);

  ggplot() +
  # geometry
  geom_sf() +
  # label
  geom_sf_text_repel(mapping = aes(label= facility_name),

                     force = 40, 
                     box.padding = 0.6)

6 Wrap up

â€¢ Adding layers on ggplot2 is relatively straightforward, as long as the data is properly stored in an sf object.

â€¢ Adding additional layers like point or polygon labels and map annotations would simply follow the same logic, with additional calls after geom_sf() and at the right place in the ggplot2 sequence.

â€¢ [scale distance](https://paleolimbot.github.io/ggspatial/reference/annotation_scale.html) is set to `"km"` by default, you can also set it in "m", "cm", "mi", "ft", or "in".
:::

## Add hospital names with `geom_sf_text_repel()`

â€¢ Use `geom_sf_text()` to add hospital locations and names as text:

```{r cities-plot}
sle_healthsites_set %>% 
  ggplot() +
  # geometry
  geom_sf() +
  # label
  geom_sf_text(mapping = aes(label= facility_name)) # ğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆ
```

â€¢ This is not really satisfactory, as the names overlap on the points, and it's not easy to read on the background.

â€¢ [`{ggsflabel}`](https://yutannihilation.github.io/ggsflabel/index.html) inspired in `{ggrepel}`deal with label placement in `ggplot` maps with `sf` objects, including automated movement of labels in case of overlap.

â€¢ `geom_sf_text_repel` and `geom_sf_label_repel`

â€¢ We "nudge" `facility_name` away, and connect them to the city locations:

```{r cities-plot-nudge}
sle_healthsites_set %>% 
  ggplot() +
  # geometry
  geom_sf() +
  # label
  geom_sf_text_repel(aes(label = facility_name)) # ğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆ
```

::: practice
`zimbabwe_hospitals` contains all the `hospital` facilities of `Zimbabwe`:

â€¢ We can use [`{ggrepel}` arguments](https://ggrepel.slowkow.com/articles/examples.html) to improve its output:

â€¢ The size (argument `size`);

â€¢ The type of font (`fontface`);

â€¢ The force of repulsion between overlapping text labels (`force`);

â€¢ The additional padding around the each text label (`box.padding`).

```{r}
sle_healthsites_set %>% 
  ggplot() +
  # geometry
  geom_sf() +
  # label
  geom_sf_text_repel(mapping = aes(label= facility_name),
                     size = 2,         # ğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆ
                     fontface = "bold",
                     force = 40, 
                     box.padding = 0.6)
```

â€¢ Adding this layer to the current map, we obtain:

```{r}
ggplot() +
  
  # country map
  geom_sf(data = sierra_leone) +
  
  # hospital points 
  geom_sf(data = sle_healthsites_set) +         
  # hospital names with repelled text 
  geom_sf_text_repel(data = sle_healthsites_set,         # ğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆ
                     mapping = aes(label = facility_name),
                     size         = 2,
                     fontface     = "bold",
                     force        = 0.5,
                     box.padding  = 0.6,
                     nudge_x      = -0.25,
                     direction    = "y",
                     hjust        = 1,
                     segment.size = 0.2) +
  
  # map annotation
  annotation_north_arrow(location="tr") +
  annotation_scale(location="bl") +
  
  # map extent
  coord_sf(xlim = c(-13.5,-12.7), 
           ylim = c(8.0,8.7))
```

# Final map

â€¢ Let's put everything together:

â€¢ Adding district boundaries filled by area, and field sited recorded by a GPS device:

```{r, message=FALSE,warning=FALSE}
# districts
sierra_leone_shp <- 
  sf::read_sf(here("data/boundaries/sle_adm3.shp")) %>% 
  filter(admin1Name=="Western")

# field site points
sites_sf <- readr::read_csv(
  here("data/gps_healthsites.csv")) %>% 
  st_as_sf(coords = c("gps_longitude","gps_latitude"), crs = 4326)
```

```{r,fig.width=8,fig.height=7}
ggplot() +
  
  # country map
  geom_sf(data = sierra_leone) +
  
  # districts filled by area
  geom_sf(data = sierra_leone_shp, # ğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆ
          mapping = aes(fill = area_km2)) +
  # color scale for fill
  scale_fill_continuous_sequential(palette="Reds 3", 
                                   alpha = 0.8) +
  # field site points
  geom_sf(data = sites_sf,              # ğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆğŸ‘ˆ
          mapping = aes(color = gps_name),
          shape = 18, size = 4) +
  
  # province names with repelled labels
  geom_sf_label_repel(data = sierra_leone %>% 
                        filter(shapeName!="Eastern"),
                      mapping = aes(label=shapeName)) +
  
  # hospital points 
  geom_sf(data = sle_healthsites_set) +         
  # hospital names with repelled text 
  geom_sf_text_repel(data = sle_healthsites_set,
                     mapping = aes(label = facility_name),
                     size         = 2,
                     fontface     = "bold",
                     force        = 0.5,
                     box.padding  = 0.6,
                     nudge_x      = -0.25,
                     direction    = "y",
                     hjust        = 1,
                     segment.size = 0.2) +
  
  # map annotation
  annotation_north_arrow(location="tl") +
  annotation_scale(location="bl") +
  
  # map extent
  coord_sf(xlim = c(-13.5,-12.7),
```

â€¢ Adding **layers** on `ggplot2` is relatively straightforward, as long as the data is properly stored in an `sf` object.

â€¢ Adding additional layers like point or polygon **labels** and **map annotations** would simply follow the same logic, with additional calls *after* `geom_sf()` and at the *right place* in the `ggplot2` sequence.
